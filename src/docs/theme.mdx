# Theme

Use Violin theme in your project (optional): 

```javascript
import { ThemeProvider } from "styled-components";
import { theme } from "violin";

<ThemeProvider theme={theme}>
  { /** root component */ }
</ThemeProvider>
```

## Custom theme

You can update an specific value from default [theme object](#base-theme-object) or create new properties and values.

```javascript
import { ThemeProvider } from "styled-components";
import { createTheme } from "violin"; 

const myCustomTheme = createTheme({
  palette: {
    primary: {
      regular: "purple" // This changes primary color only
    }
  },
  // customProperty: "some value" 
});

<ThemeProvider theme={myCustomTheme}>
  { /** root component */ }
</ThemeProvider>
```

## Global style

```javascript
import { GlobalStyle } from "violin";

const App = () => {
  return (
    <>
      <GlobalStyle />
      { /** */ }
    </>
  )
}
```
