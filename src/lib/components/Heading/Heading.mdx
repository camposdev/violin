## Use as another element

By default `Heading` is used with `h2` element, to change use `as` prop:

<Playground>
    <Heading as="h1">Lorem ipsum dolor</Heading>
    <Heading as="h3" title="sub">Lorem ipsum dolor</Heading>
    <Heading as="h4" title="section">Lorem ipsum dolor</Heading>
</Playground>
