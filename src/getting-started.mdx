# Overview

Libraries:

* [styled-components](https://www.styled-components.com/)
* [styled-tools](https://github.com/diegohaz/styled-tools)
* [polished](https://polished.js.org)

## Install 

> `styled-components` is required

Install with Yarn:

```shell
yarn add styled-component git+ssh://git@github.com:involvestecnologia/violin.git
```

Import components

```javascript
import { Button, Input } from "violin";
```
